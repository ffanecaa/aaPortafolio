Retroshop

Aplicación web e compra-venta de tecnología retro. Las funcionalidades implementadas son registro, verificado con el envio email, y login,busqueda por categoria , producto, manejo de perfil con historico ventas y compras ,opcion de favoritos , valoracion de vendedor con el promedio de esas valoraciones...

En el frontend se usó React y para los se optó por Tailwind, en la parte del backend se usó express y Msql para la base de datos.
